# 一、随机变量及其分布函数

## 1. 随机变量

在样本空间 $S$ 上的实值函数 $\mathbf X = \mathbf X(e)$，$e \in S$，称 $\mathbf X = \mathbf X(e)$ 为随机变量，简记 $\mathbf X$。

## 2. 分布函数

### (1) 定义

设 $\mathbf X$ 是一个随机变量，$x$ 是任意实数，记函数

$$
F(x) = P \{ \mathbf X \leq x \},- \infty < x < + \infty
$$

称 $F(x)$ 为 $\mathbf X$ 的分布函数。

### (2) 性质

① $F(x)$ 是一个单调不减函数，即当 $x_1 < x_2$，有 $F(x_1) ≤ F(x_2)$；

② $0 ≤ F(x) ≤ 1$，$F(- \infty)=0$，$F(+ \infty) = 1$；

③ $F(x)$ 是右连续的，即 $F(x+0)=F(x)$。
