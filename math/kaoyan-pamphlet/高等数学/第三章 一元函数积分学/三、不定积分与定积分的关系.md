# 三、不定积分与定积分的关系

## 1. 积分上限函数

设函数 $f(x)$ 在 $[a,b]$ 上连续，则 $\Phi(x) = \int^x_a f(t) \mathrm{d}t , x \in [a,b]$ 称为积分上限函数。

## 2. 积分上限函数导数

(1) 设函数 $f(x)$ 在 $[a,b]$ 上连续，则积分上限函数

$$\Phi(x) = \int^x_a f(t) \mathrm{d}t$$

在 $[a,b]$ 可导，且

$$\Phi'(x) = {\frac{\mathrm{d}}{\mathrm{d}x}} \int^x_a f(t) \mathrm{d}t = f(x) (a≤x≤b)$$

(2) 设函数 $f(x)$ 在 $[a,b]$ 上连续，则积分上限函数

$$\Phi(x) = \int^x_a f(t) \mathrm{d}t$$

是 $f(x)$ 在 $[a,b]$ 上的一个原函数，而

$$\Phi(x) + C_1 = \int^x_a f(t) \mathrm{d}t + C_1 = \int^x_0 f(t) \mathrm{d}t + C \left( C = C_1 + \int^0_a f(t) \mathrm{d}t  \right)$$

是 $f(x)$ 在 $[a,b]$ 上的所有原函数。

## 3. 牛顿—莱布尼茨公式（微积分基本公式）

设连续函数 $f(x)$ 在 $[a,b]$ 上的一个原函数 $F(x)$，则

$$
\int^b_a f(x) \mathrm{d}x = F(x) \bigg|^b_a = F(b) - F(a)
$$
