# 一、不定积分

## 1. 原函数与不定积分的定义

### 原函数

设函数 $f(x)$ 在区间 $I$ 上有定义，如果存在函数 $F(x)$，使得

$$F'(x)=f(x)或\mathrm{d}F(x)=f(x)\mathrm{d}x,x \in I$$

则称函数 $F(x)$ 为 $f(x)$ 在区间 $I$ 上的一个**原函数**。$F(x)+C$ 就是 $f(x)$ 的所有原函数。

### 不定积分

在区间 $I$ 上，$f(x)$的不定积分为 $f(x)$ 的所有原函数 $F(x)+C$，记为 $\int f(x) \mathrm{d}x$，即

$$\int f(x) \mathrm{d}x = F(x)+C$$

其中 $F(x)$ 是 $f(x)$ 的一个原函数，$C$是任意常数。

## 2. 原函数存在定理

如果函数 $f(x)$ 在区间 $I$ 上连续，则 $f(x)$ 在区间 $I$ 上存在原函数，即在区间 $I$ 上存在不定积分，故**连续函数一定存在原函数与不定积分**。

## 3. 不定积分的几何意义

如果 $f(x)$ 的一个原函数 $F(x)$，则 $y=F(x)$ 的图形称为 $f(x)$ 的一条**积分曲线**，而 $y=F(x)+C$ 的图形称为 $f(x)$ 的**积分曲线族**。

## 4. 基本积分公式

$$
\begin{align}
(1) \int k\mathrm{d}x &= kx+C (k是常数) \\\\
(2) \int x^{\mu} \mathrm{d}x &= {\frac 1 {\mu+1}} x^{\mu+1} (\mu≠-1)，特别地，\int {\frac 1 {\sqrt x}} {d}x = 2\sqrt x+C \\\\
(3) \int {\frac 1 x} \mathrm{d}x &= \ln |x| + C \\\\
(4) \int a^x \mathrm{d}x &= {\frac {a^x} {\ln a}} + C (a>0,a≠1)，特别地，\int e^x \mathrm{d}x = e^x + C \\\\
(5) \int \sin x \mathrm{d}x &= - \cos x + C \\\\
(6) \int \cos x \mathrm{d}x &= \sin x + C \\\\
(7) \int \tan x \mathrm{d}x &= - \ln |\cos x| + C \\\\
(8) \int \cot x \mathrm{d}x &= \ln |\sin x| + C \\\\
(9) \int \sec x \mathrm{d}x &= \ln |\sec x + \tan x| + C \\\\
(10) \int \csc x \mathrm{d}x &= \ln |\csc x - \cot x| + C \\\\
(11) \int sec^2x \mathrm{d}x &= \tan x + C \\\\
(12) \int csc^2x \mathrm{d}x &= - \cot x + C \\\\
(13) \int \sec x \tan x \mathrm{d}x &= \sec x + C \\\\
(14) \int \csc x \cot x \mathrm{d}x &= - \csc x + C \\\\
(15) \int {\frac 1 {\sqrt{a^2-x^2}} } \mathrm{d}x &= \arcsin \frac x a + C \\\\
(16) \int {\frac 1 {a^2+x^2} } \mathrm{d}x &= \frac 1 a \arctan \frac x a + C \\\\
(17) \int {\frac 1 {x^2-a^2} } \mathrm{d}x &= \frac 1 {2a} \ln \left| \frac {x-a} {x+a} \right| + C \\\\
(18) \int {\frac 1 {\sqrt{x^2±a^2}} } \mathrm{d}x &= \ln \left| x + \sqrt{x^2±a^2} \right| + C \\\\
\end{align}
$$

## 5. 不定积分的性质

设 $f(x)$，$g(x)$ 和 $f'(x)$ 在区间 $I$ 上连续，则

$$
\begin{align}
&(1) \frac {\mathrm{d}} {\mathrm{d}x} \int f(x) \mathrm{d}x = f(x)，\mathrm{d} \int f(x) \mathrm{d}x = f(x) \mathrm{d}x \\\\
&(2) \int f'(x) \mathrm{d}x = f(x) + C，\int \mathrm{d}f(x) = f(x)+C \\\\
&(3) \int \left[f(x)±g(x)\right] \mathrm{d}x = \int f(x) \mathrm{d}x ± \int g(x) \mathrm{d}x \\\\
&(4) \int kf(x) \mathrm{d}x = k \int f(x) \mathrm{d}x (k≠0) \\\\
\end{align}
$$
